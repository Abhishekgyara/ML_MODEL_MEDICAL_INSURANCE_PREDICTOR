# Insurance Cost Predictor
A simple Machine Learning web app that predicts medical insurance cost based on user details.
## Model: Linear Regression
## Dataset: Medical Cost Personal Dataset (Insurance dataset)
## Features
* User input form (age, BMI, children, smoker, region)
* Predict insurance cost
* Display prediction with chart visualization

## Tech Stack
* Python
* Flask
* Scikit-learn
* Pandas
* HTML / CSS
* Chart.js
